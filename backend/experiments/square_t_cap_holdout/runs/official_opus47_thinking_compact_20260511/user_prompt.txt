# 经验库 — 通用冷镦工艺规则 / General Cold-Heading Heuristics

&gt; **来源 / Provenance:** 这些规则从 `cases/` + `standards/` 共 12 个真实工艺案例
&gt; 加 `textbook_cases/` 25 个龚书第 8 章案例归纳而来,**不是教科书原文**。
&gt;
&gt; **使用方式 (重要):** 这些是**推导原则**,不是查表。LLM 应当根据零件特征
&gt; 套用原则**自己算出工位数和工序顺序**,而不是按"哪个产品类别 → 几工位"查答案。

---

## 1. 工位数推导 / Station-count derivation (principle-based)

**不要按产品类别查表。按特征数累加:**

```
station_count = 1 (下料/整形) + 1 (终镦头部) + N_features
```

其中 `N_features` 按零件特征逐项累加:

| 特征 | 何时 +1 | 触发条件 |
|---|---|---|
| **head_predeform** | 头部预镦聚料 | head_d / shank_d &gt; 1.4 (体积扩大需分两道) |
| **backward_extrusion** | 每个反挤特征 | 内六角 / 内梅花 / 凹头 / 内方 |
| **forward_extrusion** | 每个杆部减径段 | 螺纹底径减径 / 阶梯杆 / 锥杆 |
| **head_trim** | 头部修边 | 方头/T头/法兰 (因方变圆需切飞边) |
| **piercing** | 冲通孔 | 通孔件、需后续攻丝 |
| **end_chamfer** | 切尾倒锥 | 尾端 ≥ 30° 倒角 + 长杆件 (L/D &gt; 4) |
| **two_blow_upset** | 强镦粗分二次 | head_d / shank_d &gt; 2.3 (单道极限) |

**推导示例 (LLM 应当这样想,不是查表):**

例 A: 内六角圆柱头螺钉, head_d=20.7, shank_d=13.7, 螺纹底径 12.5
- head/shank = 1.51 &gt; 1.4 → +1 head_predeform
- 内六角 → +1 backward_extrusion
- 螺纹底径减径 (13.73 → 12.5) → +1 forward_extrusion
- head/shank = 1.51 &lt; 2.3 → 不需 two_blow_upset
- **stations = 1 + 1 + 3 = 5**

例 B: 六角法兰螺栓, head_d=24, shank_d=14, 无内特征
- head/shank = 1.71 &gt; 1.4 → +1 head_predeform
- 法兰 (六角变法兰需修边) → +1 head_trim
- 无内特征,无减径
- **stations = 1 + 1 + 2 = 4**

例 C: 长杆铆接螺钉, head_d=14, shank_d=4, L=80
- head/shank = 3.5 &gt; 2.3 → +1 head_predeform, +1 two_blow_upset
- 阶梯杆 (有 2 个减径段) → +2 forward_extrusion
- 尾端倒锥 → +1 end_chamfer
- **stations = 1 + 1 + 5 = 7**

---

## 2. 下料尺寸 / Blank sizing (continuous formulas)

### 直径

`blank_d` 选取目标:
- 优先选**最大杆部直径或螺纹中径附近** (减少最终 forward_extrusion 量)
- 一般 `blank_d ≈ 0.95 – 1.02 × shank_d` (略大,留 forward_extrusion 整形量)
- 当头部需要大幅镦粗 (head_d / shank_d &gt; 1.7) 可适度选小,落在 0.85 – 0.95 × shank_d

观察的真实分布: `blank_d / shank_d` ∈ [0.85, 1.05] in factory cases。

### 长度 (体积守恒)

```
V_blank = V_head + V_shank_pre_extrusion + 5–8% 余量
blank_L = V_blank / (π/4 × blank_d²)
```

实操简化:
- 短件 (成品 L &lt; 30mm): blank_L 约 = 头部体积折算 + 成品 L × 1.0
- 长螺栓 (L &gt; 50mm): blank_L 约 = 头部体积折算 + 成品 L × (1.05 – 1.10)
- 螺纹底径减径段会**伸长** 5–15%, 必须扣除

观察范围: `blank_L / product_L` ∈ [1.05, 2.5] depending on head volume share。

---

## 3. 工位顺序原则 / Station-order principles

**核心原则: 先聚料,再成形,再细节。** 不要按"标准件模板"或"异形模板"查序列。

### 排序规则

1. **第 1 站永远是 `combined` / `upsetting`** —— 切料 + 整形,工件保持柱状
2. **若需 `head_predeform`**: 紧跟第 1 站,锥形过渡为下一站终镦做准备
3. **`heading` (终镦头部)**: 头部最终外径达成
4. **`backward_extrusion`**: 一定在 heading 之后(头部已成形,才能反挤)。多个反挤按"先大后小"
5. **`forward_extrusion`**: 一般在最后 1–2 站(避免后续工序破坏精度)。多段减径按"先粗减后精修"
6. **`trimming` / `piercing`**: 永远在结构成形完成后

### 不变约束

- 反挤之前头部必须已基本成形(否则反挤压力会破坏头型)
- 正挤减径段一旦做完,**禁止**在其后做镦粗(会失稳)
- 两次同向变形(如 2 道镦粗)之间最好插一道整形或退火点(冷加工硬化)

---

## 4. 物理极限 / Physical limits (从教材抽取的硬约束)

| 约束 | 极限值 | 来源 |
|---|---|---|
| 单道镦粗比 (D_out / D_in) | ≤ 2.3 | 龚书 Ch.8 + 1模2冲教程 |
| 单道镦粗高度比 (h_in / d_in) | ≤ 2.5 (l/d 镦锻规则) | 龚书 |
| 单道镦粗截面缩短率 Ep | 材料相关,见 verifier 表 | 1模2冲教程 25.7-2 |
| 正挤减面率 | ≤ 70% (一般 50–60% 最稳) | 龚书 |
| 反挤六角孔深径比 | ≤ 4 | 龚书 |
| 反挤圆孔深径比 | ≤ 6 | 龚书 |
| 体积守恒 (blank vs final) | 0.85 – 1.45 | 真实工厂案例统计 |

**关键认知**: 这些是材料物理硬约束,**不是经验软建议**。verifier 强制检查。

## 5. 后处理 / Post-processes (按特征,非按类别)

| 触发特征 | 必加后处理 |
|---|---|
| 任何外螺纹 | `thread_rolling` |
| 任何强度等级标注 ≥ 8.8 | `heat_treatment` |
| 任何内螺纹孔(攻丝孔) | `thread_tapping` (在 piercing 后) |
| 表面镀层 (Zn, Ni, 磷化) | 客户图标注才加,与冷镦无关 |

**不要按"产品类别"决定后处理**,按特征。

## 6. 材料适用性 / Material applicability

冷镦钢的工艺极限**几乎与牌号无关** —— 只要是合格冷镦钢,镦粗比 ≤ 2.3 通用。
材料牌号主要影响:
- **变形抗力**(影响模具寿命,不影响工序设计)
- **允许累积变形量** (Ep, 在 verifier 里有表)
- **是否需热处理** (高强度等级才需)

直接照抄客户图的材料牌号,**不要**因为材料推断工序数。

---

## 应用到 Step 3 的明确指令

LLM 在 Step 3 应按以下顺序思考:

1. **数特征**: 头部形状、内特征(六角/梅花/凹头)、杆部减径段数、尾端倒锥
2. **算 head/shank 比**: 决定要不要 head_predeform 和是不是 two_blow_upset
3. **套 §1 公式**: stations = 1 + 1 + N_features
4. **按 §3 顺序原则**排站
5. **按 §2 体积守恒**算 blank
6. **按 §4 物理极限**自检 (verifier 还会再检一遍)
7. **`cited_case_ids` 引用 2–3 个具有相似特征的案例** (不一定要同产品类别;
   可以是"它的反挤工序参考自 X","它的预镦比例参考自 Y")

**反模式 (不要做):**
- ❌ "这是某某标准件 → N 站" (查表)
- ❌ "我没见过这种件,所以保守用 6 站" (没推导)
- ❌ "引用最像的整件案例" (整件常找不到近邻;按特征引用更可靠)

## 预成形设计原则

p21-p23 的预成形案例给出的重点不是固定公式，而是设计思路：

- 预成形要同时保证纵向稳定性和终成形后的最佳形状。
- 毛坯细长时，先做过渡预成形，避免一次终镦造成弯曲、折叠或充不满。
- 对头部体积大、杆部细的零件，预镦形状比最终形状更重要；不要直接从圆柱坯跳到复杂最终头型。

p22-p23 里可见的工程判断阈值：

- 固定冲奶头模示例中，长度/直径比约 `2.76`，小于 `3.25`，属于可接受范围。
- 对更严格或带浮动机构的设计，实际可采用更保守的 `L/D ≈ 2.5` 限制。
- 当头部局部直径扩大接近或超过 `D1/D ≈ 1.4`，应考虑浮动模、分步成形或更保守的预成形。

应用到 过模图：

## 使用优先级

- Tier 1 工厂/标准件 case 仍是最高权重 answer key。
- 本文件用于补足工艺物理、材料准备、螺纹坯径、机器能力和后处理提醒。
- 低置信表格不要直接当尺寸答案；若与真实 case 冲突，以真实 case 为准。
- p69-p104 主要是标准模具图册页，已识别出 GB/T5784 M8x80 螺栓、GB/T70.1 M8x18 内六角圆柱头螺钉等标准案例；当前先作为待抽取 textbook cases，不直接进入硬规则。

## 毛坯与螺纹坯径

p45-p50 给出 GB/T 18685-2002 滚压普通螺纹前毛坯直径规则。

- 外螺纹不是直接取公称直径；滚牙前坯径应略小于公称大径，并接近滚压后体积守恒的中径/底径折算结果。
- 若图面或 case 已给出线径/搓牙坯径，优先使用图面/case 数值。

**推导公式（无图面数据时使用，不要查表）:**

```
滚牙前坯径 (d_thread_blank) ≈ d_nominal − 0.65 × P    (P 为螺距)
   M16×2.0 → 16 − 1.30 = 14.70  (公式为保守估算，实做按搓丝样件修正)
   M20×2.5 → 20 − 1.63 = 18.37
   M22×2.5 → 22 − 1.63 = 20.37
   M8×1.25 → 8  − 0.81 = 7.19   (与 GB/T70.1 size table 中 d1=7.10 接近)

正挤前线径 (d_blank) ≈ d_thread_blank + 0.2~0.5  (留正挤减径量)
   或者 ≈ shank_diameter (光杆段),取较大者

自攻螺纹 (Delta PT / Taptite / PowerLock)：
  滚牙前坯径 ≈ 公称大径 − 0.4 ~ 0.5
  (滚牙后径向膨胀至公称值，所以坯径必须 &lt; 公称)
```

- 车削毛坯需要考虑表面粗糙度余量；冷镦线材/拉拔线材可直接按公式或 factory case。
- 生成 `ProcessForming` 时，正挤/缩径站应明确写出 `thread_blank_D` 和 `thread_L`，并把 `thread_rolling` 放入 post_processes。

应用到 demo：

- 有外螺纹时，最终冷镦站通常不是“成品螺纹”，而是“搓牙坯径/螺纹坯段”。
- 若模型把最终杆径写成公称螺纹大径，应降置信度或要求重试。

## 内螺纹和通孔件

p51-p56 包含内螺纹底孔/攻丝孔相关表格，OCR 数字置信度低，但原则可用：

- 内螺纹牙型高度通常按理论牙高的 `70%-85%` 选择，不追求 100% 满牙。
- 内螺纹件、铆螺母、四方通孔件应显式包含：
  - 预成形/镦粗聚料
  - 冲孔或通孔成形
  - 后续攻丝/滚压内螺纹说明
- 冲孔/通孔不要和大变形终镦硬塞在同一站，除非有非常相似 factory case。

应用到 demo：

- `through_hole`、`internal_thread`、`四方通孔`、`T帽` 类零件，应优先引用四方T帽、四方通孔、铆接/铆螺母类 case。
- `post_processes` 对内螺纹可使用 `thread_rolling` 近似表示后续成牙，但 reasoning_zh 必须写明“后续攻丝/成内螺纹”。

## 变形力、机器能力和工位数

p33-p44 重复出现冷镦力/切断力/机器能力判断，数值表只作参考，原则如下：

- 切断力近似与线材截面积相关：`Pq ∝ d0^2 * τs`。
- 成形力与材料强度、头部最大投影面积、摩擦和封闭程度相关；大头、小杆、封闭反挤、内孔件需要更保守。
- 机器能力应至少覆盖估算成形力并留安全系数，工程上可按 `1.2-1.5×` 留余量。
- 最大切断直径通常要比设计线径留出余量；机器能力不足时，不应靠减少工位硬压。
- 工位数由工艺变形图决定：
  - 标准六角头：切料 -&gt; 初镦 -&gt; 终镦 -&gt; 切边。
  - 凸缘/大头件：切料 -&gt; 初镦 -&gt; 二镦/分配体积 -&gt; 终镦 -&gt; 切边。
  - 内六角圆柱头：切料/整形 -&gt; 预镦 -&gt; 终镦圆柱头 -&gt; 反挤内六角 -&gt; 正挤螺纹坯。
  - 通孔/内螺纹件：预成形和冲孔/通孔应分步体现。

应用到 demo：

- 若客户图或图框写着 4 模，但经验库同类 case 是 5 工位，可把“切料/整形”作为 station 1 单独画出，同时在 reasoning_zh 解释：客户机台工位与过模图展示站序不完全一一等同。
- 对标准件 demo，宁可输出工程上稳妥的 4-5 站，不要为了显得简单而压成 2-3 站。

## 模具/成形几何对过模图的影响

p33-p44 包含较多哈夫模、冲模、模具行程和锥角内容。当前 v2 不输出模具图，但可以影响过模图站序：

- 大直径差、深孔、异形头、方/六角切边件，通常需要把“体积分配”和“最终外形/冲孔/切边”拆开。
- 引入斜角/圆角过渡能减少折叠、充不满和应力集中；过模图里可用 `notes_zh` 标注“锥形过渡/圆角过渡”。
- 最终校形/切边站应保留关键外形尺寸，尤其是四方头、六角头、法兰边、花形外轮廓。
- p33 的哈夫模拆分、过盈、模套硬度等属于 Phase 2 tooling 参考；不要让它驱动当前 DXF 过模图坐标。

应用到 demo：

- `WorkpieceGeometry.extra_dims_mm` 可记录 `chamfer_deg`、`fillet_R`、`socket_AF`、`socket_depth`、`hole_D` 等关键特征。
- 过模图 DXF 应优先表达工件中间形态，而不是模芯、模套、冲棒结构。

## 材料软化、表面处理和润滑

p57-p68 对当前 demo 非常有用：它解释为什么冷镦/冷挤压前必须关注硬度、磷化和润滑。

### 软化处理

- 冷挤压/大变形前，毛坯软化的目的：降低硬度、提高塑性、消除残余应力、改善组织。
- 中高碳钢或合金钢如果硬度偏高，应在 reasoning_zh 标注“需球化退火/软化处理后冷镦”。
- 不锈钢、铝合金、铜/黄铜的软化和表面处理路径不同，不要默认全部按碳钢磷化处理。

### 钢件磷化和皂化

- 钢/合金钢冷镦前常用：除油 -&gt; 酸洗 -&gt; 磷化 -&gt; 中和 -&gt; 皂化/润滑。
- 磷化膜的作用：承载润滑剂、隔离金属与模具、降低摩擦、减少粘模和表面拉伤。
- 文中给出方向性数据：磷化后油吸附能力显著提高，摩擦系数可大幅降低。具体数值只作趋势，不用于硬计算。
- 大头、小杆、反挤内六角、深孔/通孔件，应在 reasoning_zh 中标注“需良好磷化皂化/润滑”。

### 不锈钢和有色金属

- 奥氏体不锈钢不能按普通碳钢磷化路径处理；应考虑草酸盐/专用表面处理和更保守工位。
- 铝合金通常通过氧化/磷化类膜层承载润滑剂，避免挤压裂纹。
- 铜/黄铜使用钝化/润滑路线；不要把钢件磷化参数直接套用。

应用到 demo：

- 材料未知时，按中等冷镦钢保守处理，confidence 不给 high，除非有同类 factory case。
- 材料为不锈钢、铝、铜、黄铜时，减少单站变形量，并在 reasoning_zh 标出材料准备风险。
- `post_processes` 可增加 `annealing` 或 `phosphating`，但不要把表面处理画成过模图 station。

## 可执行校验建议

这些规则应进入 `ProcessFormingVerifier` 或 prompt retry：

- 外螺纹件：必须有 `thread_rolling` 或明确“后续滚牙/搓牙”。
- 内六角/凹孔件：必须有 `backward_extrusion` 或明确“反挤/压内孔”。
- 通孔/内螺纹件：必须有 `piercing` 或明确“冲孔/通孔”，并有后续攻丝/成牙说明。
- 大头小杆、法兰、四方/六角切边件：如果单站最大直径增长超过约 `1.6×`，要求增加预镦/二镦。
- 材料为不锈钢/铝/铜时，若 station_count 过少或无润滑/退火说明，降 confidence。
- 只要用到 textbook-only 规则而没有相似 Tier 1 case，confidence 最高只能是 medium。

## 镦粗率与冲击次数

p1-p2 给出了镦粗率定义和冲击次数判断。Demo 中可用以下规则：

- 镦粗率可按高度变化估算：`E = (H0' - HN') / H0' * 100%`。
- 若 `l / dm &lt;= 2.8`，通常可以一次冲击。
- 若 `l / dm = 3.0-5.5`，通常需要二次冲击。
- 若 `l / dm = 5.6-8.0`，通常需要三次冲击。

多次冲击的高度分配：

- 第一次冲击后缩短到约 `75%-80% l`
- 第二次冲击后缩短到约 `40%-55% l`
- 最后一次冲击到最终高度 `hk`

应用到 过模图：

- 头部成形如果需要二次冲击，应在 ProcessForming 中体现为“预镦/初镦”和“终镦/成形头部”两个站。
- 细长杆件和大头件不要把全部镦粗量塞到一个 station。

## 头部毛坯长度估算

p3 表 25.7-3 给出不同头型的毛坯长度计算。当前 demo 先使用这些作为趋势规则：

- 平顶圆柱头：头部毛坯长度与 `D^2 * h / dm^2` 成正比。
- 六角头：可用 `1.33 * S^2 * h / dm^2` 估算头部毛坯长度。
- 四方头：可用 `1.27 * S^2 * h / dm^2` 估算头部毛坯长度。
- 沉头/半沉头/半圆头需要考虑头部轮廓体积，不能只按圆柱体积估算。
- 六角螺母毛坯直径：小规格约 `1.1-1.2 dn`，M8-M24 约 `1.0-1.1 dn`；长度按体积折算。

应用到 过模图：

- 头型不同，blank_L 不能只按成品总长拍脑袋。
- 六角头、四方头、圆柱头应按头部体积单独估算，再加杆部/螺纹坯体积。

## 冷镦力快速估算

p4 给出冷镦力和变形力影响因素。Demo 阶段的简化判断：

- 冷镦力与头部最大直径的平方强相关。
- 变形程度在 `50%-60%` 时，经验系数常取 `5-6` 量级。
- 变形方式、润滑、模具结构、材料机械性能都会改变所需吨位。

应用到 过模图：

- 大头、小杆、封闭成形、反挤内孔都应降低单站变形量。
- 这类零件的 confidence 不应轻易给 high，除非有非常相似的 worked case。
