## 毛坯与螺纹坯径

p45-p50 给出 GB/T 18685-2002 滚压普通螺纹前毛坯直径规则。

- 外螺纹不是直接取公称直径；滚牙前坯径应略小于公称大径，并接近滚压后体积守恒的中径/底径折算结果。
- 若图面或 case 已给出线径/搓牙坯径，优先使用图面/case 数值。

**推导公式（无图面数据时使用，不要查表）:**

```
滚牙前坯径 (d_thread_blank) ≈ d_nominal − 0.65 × P    (P 为螺距)
   M16×2.0 → 16 − 1.30 = 14.70  (公式为保守估算，实做按搓丝样件修正)
   M20×2.5 → 20 − 1.63 = 18.37
   M22×2.5 → 22 − 1.63 = 20.37
   M8×1.25 → 8  − 0.81 = 7.19   (与 GB/T70.1 size table 中 d1=7.10 接近)

正挤前线径 (d_blank) ≈ d_thread_blank + 0.2~0.5  (留正挤减径量)
   或者 ≈ shank_diameter (光杆段),取较大者

自攻螺纹 (Delta PT / Taptite / PowerLock)：
  滚牙前坯径 ≈ 公称大径 − 0.4 ~ 0.5
  (滚牙后径向膨胀至公称值，所以坯径必须 < 公称)
```

- 车削毛坯需要考虑表面粗糙度余量；冷镦线材/拉拔线材可直接按公式或 factory case。
- 生成 `ProcessForming` 时，正挤/缩径站应明确写出 `thread_blank_D` 和 `thread_L`，并把 `thread_rolling` 放入 post_processes。

应用到 demo：

- 有外螺纹时，最终冷镦站通常不是“成品螺纹”，而是“搓牙坯径/螺纹坯段”。
- 若模型把最终杆径写成公称螺纹大径，应降置信度或要求重试。

## 内螺纹和通孔件

p51-p56 包含内螺纹底孔/攻丝孔相关表格，OCR 数字置信度低，但原则可用：

- 内螺纹牙型高度通常按理论牙高的 `70%-85%` 选择，不追求 100% 满牙。
- 内螺纹件、铆螺母、四方通孔件应显式包含：
  - 预成形/镦粗聚料
  - 冲孔或通孔成形
  - 后续攻丝/滚压内螺纹说明
- 冲孔/通孔不要和大变形终镦硬塞在同一站，除非有非常相似 factory case。

应用到 demo：

- `through_hole`、`internal_thread`、`四方通孔`、`T帽` 类零件，应优先引用四方T帽、四方通孔、铆接/铆螺母类 case。
- `post_processes` 对内螺纹可使用 `thread_rolling` 近似表示后续成牙，但 reasoning_zh 必须写明“后续攻丝/成内螺纹”。

## 变形力、机器能力和工位数

p33-p44 重复出现冷镦力/切断力/机器能力判断，数值表只作参考，原则如下：

- 切断力近似与线材截面积相关：`Pq ∝ d0^2 * τs`。
- 成形力与材料强度、头部最大投影面积、摩擦和封闭程度相关；大头、小杆、封闭反挤、内孔件需要更保守。
- 机器能力应至少覆盖估算成形力并留安全系数，工程上可按 `1.2-1.5×` 留余量。
- 最大切断直径通常要比设计线径留出余量；机器能力不足时，不应靠减少工位硬压。
- 工位数由工艺变形图决定：
  - 标准六角头：切料 -> 初镦 -> 终镦 -> 切边。
  - 凸缘/大头件：切料 -> 初镦 -> 二镦/分配体积 -> 终镦 -> 切边。
  - 内六角圆柱头：切料/整形 -> 预镦 -> 终镦圆柱头 -> 反挤内六角 -> 正挤螺纹坯。
  - 通孔/内螺纹件：预成形和冲孔/通孔应分步体现。

应用到 demo：

- 若客户图或图框写着 4 模，但经验库同类 case 是 5 工位，可把“切料/整形”作为 station 1 单独画出，同时在 reasoning_zh 解释：客户机台工位与过模图展示站序不完全一一等同。
- 对标准件 demo，宁可输出工程上稳妥的 4-5 站，不要为了显得简单而压成 2-3 站。

## 模具/成形几何对过模图的影响

p33-p44 包含较多哈夫模、冲模、模具行程和锥角内容。当前 v2 不输出模具图，但可以影响过模图站序：

- 大直径差、深孔、异形头、方/六角切边件，通常需要把“体积分配”和“最终外形/冲孔/切边”拆开。
- 引入斜角/圆角过渡能减少折叠、充不满和应力集中；过模图里可用 `notes_zh` 标注“锥形过渡/圆角过渡”。
- 最终校形/切边站应保留关键外形尺寸，尤其是四方头、六角头、法兰边、花形外轮廓。
- p33 的哈夫模拆分、过盈、模套硬度等属于 Phase 2 tooling 参考；不要让它驱动当前 DXF 过模图坐标。

应用到 demo：

- `WorkpieceGeometry.extra_dims_mm` 可记录 `chamfer_deg`、`fillet_R`、`socket_AF`、`socket_depth`、`hole_D` 等关键特征。
- 过模图 DXF 应优先表达工件中间形态，而不是模芯、模套、冲棒结构。

## 材料软化、表面处理和润滑

p57-p68 对当前 demo 非常有用：它解释为什么冷镦/冷挤压前必须关注硬度、磷化和润滑。

### 软化处理

- 冷挤压/大变形前，毛坯软化的目的：降低硬度、提高塑性、消除残余应力、改善组织。
- 中高碳钢或合金钢如果硬度偏高，应在 reasoning_zh 标注“需球化退火/软化处理后冷镦”。
- 不锈钢、铝合金、铜/黄铜的软化和表面处理路径不同，不要默认全部按碳钢磷化处理。

### 钢件磷化和皂化

- 钢/合金钢冷镦前常用：除油 -> 酸洗 -> 磷化 -> 中和 -> 皂化/润滑。
- 磷化膜的作用：承载润滑剂、隔离金属与模具、降低摩擦、减少粘模和表面拉伤。
- 文中给出方向性数据：磷化后油吸附能力显著提高，摩擦系数可大幅降低。具体数值只作趋势，不用于硬计算。
- 大头、小杆、反挤内六角、深孔/通孔件，应在 reasoning_zh 中标注“需良好磷化皂化/润滑”。

### 不锈钢和有色金属

- 奥氏体不锈钢不能按普通碳钢磷化路径处理；应考虑草酸盐/专用表面处理和更保守工位。
- 铝合金通常通过氧化/磷化类膜层承载润滑剂，避免挤压裂纹。
- 铜/黄铜使用钝化/润滑路线；不要把钢件磷化参数直接套用。

应用到 demo：

- 材料未知时，按中等冷镦钢保守处理，confidence 不给 high，除非有同类 factory case。
- 材料为不锈钢、铝、铜、黄铜时，减少单站变形量，并在 reasoning_zh 标出材料准备风险。
- `post_processes` 可增加 `annealing` 或 `phosphating`，但不要把表面处理画成过模图 station。

## 可执行校验建议

这些规则应进入 `ProcessFormingVerifier` 或 prompt retry：

- 外螺纹件：必须有 `thread_rolling` 或明确“后续滚牙/搓牙”。
- 内六角/凹孔件：必须有 `backward_extrusion` 或明确“反挤/压内孔”。
- 通孔/内螺纹件：必须有 `piercing` 或明确“冲孔/通孔”，并有后续攻丝/成牙说明。
- 大头小杆、法兰、四方/六角切边件：如果单站最大直径增长超过约 `1.6×`，要求增加预镦/二镦。
- 材料为不锈钢/铝/铜时，若 station_count 过少或无润滑/退火说明，降 confidence。
- 只要用到 textbook-only 规则而没有相似 Tier 1 case，confidence 最高只能是 medium。
