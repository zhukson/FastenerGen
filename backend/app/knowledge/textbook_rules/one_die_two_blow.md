## 定位

这份教程对当前 demo 的价值主要在两个地方：

- 给标准件和少工位零件提供基础工艺模板。
- 给 Step 3 一个物理 sanity check：毛坯允许镦粗率、冲击次数、头部毛坯长度、典型成形顺序。

当前 demo 只输出 过模图，不输出冲模/凹模工程图；模具材料与硬度表先作为 Phase 2 参考，不进入主要 prompt 判断。

## 线材尺寸公差

目视抽取自 p1 表 25.7-1：

| 线材外径 | 公差 | 圆度 |
|---|---:|---:|
| `<9.00` | `+0 / -0.03` | `<0.015` |
| `9.00-18.00` | `+0 / -0.04` | `<0.02` |
| `>18.00` | `+0 / -0.05` | `<0.025` |

应用到 过模图：

- LLM 不要把线径写得比客户图更精细；可保留一位或两位小数。
- 标准件线径通常略小于公称螺纹外径，用于后续滚牙。

## 镦粗率与冲击次数

p1-p2 给出了镦粗率定义和冲击次数判断。Demo 中可用以下规则：

- 镦粗率可按高度变化估算：`E = (H0' - HN') / H0' * 100%`。
- 若 `l / dm <= 2.8`，通常可以一次冲击。
- 若 `l / dm = 3.0-5.5`，通常需要二次冲击。
- 若 `l / dm = 5.6-8.0`，通常需要三次冲击。

多次冲击的高度分配：

- 第一次冲击后缩短到约 `75%-80% l`
- 第二次冲击后缩短到约 `40%-55% l`
- 最后一次冲击到最终高度 `hk`

应用到 过模图：

- 头部成形如果需要二次冲击，应在 ProcessForming 中体现为“预镦/初镦”和“终镦/成形头部”两个站。
- 细长杆件和大头件不要把全部镦粗量塞到一个 station。

## 头部毛坯长度估算

p3 表 25.7-3 给出不同头型的毛坯长度计算。当前 demo 先使用这些作为趋势规则：

- 平顶圆柱头：头部毛坯长度与 `D^2 * h / dm^2` 成正比。
- 六角头：可用 `1.33 * S^2 * h / dm^2` 估算头部毛坯长度。
- 四方头：可用 `1.27 * S^2 * h / dm^2` 估算头部毛坯长度。
- 沉头/半沉头/半圆头需要考虑头部轮廓体积，不能只按圆柱体积估算。
- 六角螺母毛坯直径：小规格约 `1.1-1.2 dn`，M8-M24 约 `1.0-1.1 dn`；长度按体积折算。

应用到 过模图：

- 头型不同，blank_L 不能只按成品总长拍脑袋。
- 六角头、四方头、圆柱头应按头部体积单独估算，再加杆部/螺纹坯体积。

## 冷镦力快速估算

p4 给出冷镦力和变形力影响因素。Demo 阶段的简化判断：

- 冷镦力与头部最大直径的平方强相关。
- 变形程度在 `50%-60%` 时，经验系数常取 `5-6` 量级。
- 变形方式、润滑、模具结构、材料机械性能都会改变所需吨位。

应用到 过模图：

- 大头、小杆、封闭成形、反挤内孔都应降低单站变形量。
- 这类零件的 confidence 不应轻易给 high，除非有非常相似的 worked case。

## 典型零件冷镦工艺

目视抽取自 p4-p5 表 25.7-5：

| 零件 | 典型成形工序 |
|---|---|
| 六角头螺栓 | 切料 -> 初镦 -> 终镦 -> 切边 |
| 凸缘螺栓 | 切料 -> 初镦 -> 二镦 -> 终镦 -> 切边 |
| 钢球 | 切料 -> 镦球 -> 光球 |
| 内六角螺钉 | 切料 -> 初镦 -> 二镦 -> 内六角孔成形 |
| 半圆头方颈螺栓 | 切料 -> 初镦 -> 终镦 -> 切边 |
| 连杆螺栓 | 切料 -> 整形 -> 初镦 -> 二镦 -> 终镦 |
| 六角螺母 | 切料 -> 镦球/预成形 -> 镦六角 -> 冲孔 |
| 阀门弹簧座圈 | 切料 -> 整形 -> 镦球 -> 冲孔 -> 退火磷化 -> 终镦 |

应用到 Step 3：

- DIN933 可优先套六角头螺栓模板。
- DIN912 可优先套内六角螺钉模板，但本项目现有标准件 cases 显示通常还会包含正挤螺纹坯/杆部调整。
- 六角螺母和空心件一定要显式出现冲孔/通孔步骤。
